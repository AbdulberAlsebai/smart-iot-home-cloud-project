# Smart IoT Home System – Cloud Computing Project

This university project presents a scalable Smart IoT Home System architecture built using modern cloud computing technologies.

## Technologies & Concepts
- AWS Cloud Services
- Kubernetes (EKS)
- Docker Containers
- Microservices Architecture
- Public & Private Subnets
- Load Balancing
- IAM & Security
- IoT Integration
- CAP Theorem
